Features

React ist eine Javascript-Bibliothek zur deklarativen

JSX

Bei JSX handelt es sich um eine Syntaxerweiterung für Javascript, die es ermöglicht, „HTML in Javascript“ zu schreiben. Damit werden HTML-Tags zu validen Werten beispielsweise für Variablen oder die Rückgabe von Funktionen.

Da class und for in Javascript bereits reservierte Wörter sind, sollte auf className respektive htmlFor ausgewichen werden. Javascript-Ausdrücke können von geschweiften Klammern umgeben eingegeben werden. Zur Verhinderung von Cross-Site-Scripting (XSS) werden diese Ausdrücke escaped. [4]

Um JSX verwenden zu können, muss React importiert sein und ein Transpiler, zum Beispiel Babel mit Webpack, eingerichtet sein. Das Kommandozeilen-Werkzeug create-react-app, [2] welches zum initialen Erstellen einer React-Anwendung empfohlen wird, erledigt die Einrichtung von Webpack bereits.

Babel überführt dann den JSX-Code, beispielsweise

<button type="submit">Senden</button>

in gewöhnlichen Javascript-Code:

React.createElement(
    "button",
    { type: "submit" },
    "Senden"
);
[3]

Man sieht hier, dass JSX zu lesbarerem Code verhelfen kann. Allerdings ist es ebenso möglich, auf JSX und damit auch auf Babel zu verzichten. Die Verwendung von JSX ist optional.

Komponenten

Komponenten sind wiederverwendbare und voneinander isolierte Objekte, die die React-Bibliothek auf das Document Object Model (DOM) abbildet.
